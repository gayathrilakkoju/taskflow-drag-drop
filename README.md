# TaskFlow – Drag & Drop Task Manager

## Description
TaskFlow is a simple web application where users can add, move, and delete tasks. Tasks can be organized into different stages like To Do, In Progress, and Done.

##  Features
- Add new tasks
- Drag and drop tasks between columns
- Delete tasks
- Tasks are saved even after refreshing the page

## 🛠️ Technologies
- HTML
- CSS
- JavaScript

## How It Works
Tasks are stored in the browser using localStorage, so they remain available even after page reload.
